JavaScript variables can be objects. Arrays are special kinds of objects. You can have variables of different types, function, arrays or other objects in an Array.


Array Methods

JavaScript Methods are Actions that can be performed on objects. A JavaScript Method is a property containing a function definition


Looping Through Arrays

The safest way to loop through an array is a 'for' loop, usually written as such:

for (i = 0; i < arrayName.length; i++)

You can also use the Array.forEach() function:

arrayName.forEach(functionName), where the function is what you want to do with the data. This example logs the `consoles` array variables into an unordered list of `value`s.


Adding Array Elements

The easiest way to add a new element to an array is using the `.push()` method:

arrayName.push("value");

The push method appends a new element to an array.


New elements can also be added to an array using the `.length` property:

arrayName[arrayName.length] = "value";

The `.length` property provides an easy way to append new elements to an array without using the `push()` method.
